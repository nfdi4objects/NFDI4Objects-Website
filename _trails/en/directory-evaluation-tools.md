## Project Description

This trail project focuses on the generation of documentation data during field research, archiving, and the intermediate work steps.

## Goal

Creation of a directory of software tools, standards, and their evaluation for archaeological research.

## Work Steps

- Survey of existing tools
- Documentation of standards
- Evaluation of tools
- Publication of the directory
